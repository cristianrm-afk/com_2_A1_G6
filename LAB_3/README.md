# Laboratorio de Radiofrecuencia y Envolvente Compleja



En este repositorio se encuentra el desarrollo de la práctica 3 de Comunicaciones II, enfocada en el análisis de señales de radiofrecuencia y su representación mediante envolvente compleja en GNU Radio.



Durante la práctica se trabajó con modulaciones OOK, BPSK y FSK, realizando observaciones en el dominio del tiempo, frecuencia y constelación, con el fin de comparar el comportamiento de la señal en versión RF y en banda base.



Herramientas utilizadas

GNU Radio

Git y GitHub

Ubuntu / Linux

Contenido

Archivos .grc de las simulaciones

Informe de la práctica

Evidencias del proceso

Objetivo



Comprender la diferencia entre una señal modulada en RF y su equivalente en envolvente compleja.
