# com_2_A1_G6
Comunicaciones II - UIS

Repositorio con prácticas, simulaciones y proyectos desarrollados en la asignatura Comunicaciones II de la Universidad Industrial de Santander (UIS).

Contenido
Modulación digital (QAM, PSK, FSK)
Análisis de señales en banda base y pasabanda
Simulaciones en MATLAB / Python
Ruido y desempeño de sistemas de comunicación

Herramientas
MATLAB / Simulink
Python (NumPy, SciPy, Matplotlib)

Objetivo
Aplicar conceptos teóricos de comunicaciones para el análisis y diseño de sistemas modernos.
